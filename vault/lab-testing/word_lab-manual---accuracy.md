## Accuracy


People use the terms accuracy and precision interchangeably, but in fact they represent two very different things. Accuracy is a measure of how close a test result is to the “true” value, while precision is a measure of how closely repeated measurements of the same sample agree with each other. For individual analyses we cannot measure the accuracy, but we can measure the precision. Instead of measuring the accuracy of an individual analysis, we test the accuracy of method itself, through a process called validation.

Accuracy is a measure of how close a result is to the “true” value. But this is a Catch-22. To know the accuracy of our measurement, the true value must be known. If the true value is known, then we don’t need to measure it. If the true value is not known, then how can we know how close our result is to that true value? In fact, the reason the test is being done in the first place is because the value is unknown.

So how do we know how close our result is to the ‘true’ value? This is where an important concept of analysis comes into play: validation. Validation is a process to test the method to measure the accuracy of results generated using that method. When you validate a method, you test samples with known amounts of analytes to determine what percentage of the known value is recovered using that method. That is, how close is the result to the known value?

Each lab must decide for itself what an acceptable accuracy rate is, and it may differ depending upon the method. Most validation schemes will accept recovery of between 95% and 105%. For titratable acidity (TA), for example, this would mean a result of 5.0 g/L malic acid means the “true” value is somewhere between 4.8 g/L and 5.2 g/L. Note that this may be acceptable range for TA, however it may not be an acceptable range for alcohol. In this case, a method that gives recovery of 95% to 105% means a result of 7.0 % v/v alcohol represents a “true” value somewhere between 6.6% and 7.4% v/v alcohol. For most cidermakers, this range of accuracy would be unacceptable.
